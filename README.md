# functions

## 介绍
- 京东城市前端工具函数库

## 开发
### clone项目
- `git clone http://git.jd.com/domain/functions.git`
- `cd functions`
- `npm install`

### 开发一个feature
- `git checkout -b feature-<feature name>`
- coding
- `npm run test`
- `git push origin feature-<feature name>`
- 提交 merge request
- merge to master

## 使用
> `npm install -S git+http://domain.git`
